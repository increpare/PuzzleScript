(*** Gameplay. ***)

(Planting)
[ action Player GrassCarried Arable no Blocker no GrowingCrops ] -> [ Player Arable GrassYoung ]
[ action Player ReedCarried Shore no Blocker no GrowingCrops ] -> [ Player Shore ReedYoung ]
[ action Player FlowerCarried Arable no Blocker no GrowingCrops ] -> [ Player Arable FlowerStraight GrowthParticles1 ]
[ action Player KumaraCarried Arable no Blocker no GrowingCrops ] -> [ Player Arable KumaraYoung ]
[ action Player TomatoCarried Arable no Blocker no GrowingCrops ] -> [ Player Arable TomatoYoung ]
[ action Player MushroomCarried Arable no Blocker no GrowingCrops ] -> [ Player Arable MushroomLargeStraight MushroomChanged random RandomGrowthParticles1 ]
[ action Player TreeCarried TreeGoal TreeGoalMarker no Blocker no GrowingCrops ] -> [ Player TreeGoal TreeBase random TreeParticles1 ]

up [ TreeBase | ] -> [ TreeBase | TreeCrownMiddle ]
right [ | TreeCrownMiddle | ] -> [ TreeCrownLeft | TreeCrownMiddle | TreeCrownRight ]

(Putting back)
[ action Player GrassCarried Stone no Resource ] -> [ Player GrassResource Stone ]
[ action Player ReedCarried Stone no Resource ] -> [ Player ReedResource Stone ]
[ action Player FlowerCarried Stone no Resource ] -> [ Player FlowerResource Stone ]
[ action Player KumaraCarried Stone no Resource ] -> [ Player KumaraResource Stone ]
[ action Player TomatoCarried Stone no Resource ] -> [ Player TomatoResource Stone ]
[ action Player MushroomCarried Stone no Resource ] -> [ Player MushroomResource Stone ]
[ action Player TreeCarried Stone no Resource ] -> [ Player TreeResource Stone ]

(Is the exit ready, all tree goals filled with a base?)
[ ExitLeft ] -> [ ExitLeft ExitLeftIndicator ]
[ ExitRight ] -> [ ExitRight ExitRightIndicator ]
[ ExitUp ] -> [ ExitUp ExitUpIndicator ]
[ ExitDown ] -> [ ExitDown ExitDownIndicator ]
[ HouseDoorShut ] -> [ HouseDoorShut ExitUpIndicator ]
[ Exit ExitIndicator ] [ TreeGoal no TreeBase ] -> [ Exit ] [ TreeGoal ]
[ HouseDoorShut ExitIndicator ] [ TreeGoal no TreeBase ] -> [ HouseDoorShut ] [ TreeGoal ]
(Repair appropriate bridges.)
[ ExitIndicator ] [ RepairOnExitReady HorizontalBridgeBroken ] -> [ ExitIndicator ] [ HorizontalBridgeRepairing1 ]
[ ExitIndicator ] [ RepairOnExitReady VerticalBridgeBroken ] -> [ ExitIndicator ] [ VerticalBridgeRepairing1 ]
[ HouseDoorShut ExitIndicator ] -> [ HouseDoorOpen ExitUp ]
[ HouseDoorOpen ExitIndicator ] -> [ HouseDoorOpen ]

(Grass growing and spreading)
[ _SP ] [ GrassYoung ] -> [ _SP ] [ GrassMatureStraight random RandomGrowthParticles1 ]
[ _SP ] [ GrassMature | Arable no Blocker ] -> [ _SP ] [ GrassMature | Arable GrassYoung ]

(Kumara growing and spreading horizontally)
[ _SP ] [ KumaraYoung ] -> [ _SP ] [ KumaraMatureStraight random RandomGrowthParticles1 ]
horizontal [ _SP ] [ KumaraMature | Arable no Blocker ] -> [ _SP ] [ KumaraMature | Arable KumaraYoung ]

(Mushrooms spreading a bit differently. Minis grow into large if they're touching 2+ large.)
[ _SP ] [ MushroomLarge | MushroomMini | MushroomLarge ] -> [ _SP ] [ MushroomLarge | MushroomLargeStraight MushroomChanged random RandomGrowthParticles1 | MushroomLarge ]
horizontal [ _SP ] [ MushroomLarge | MushroomMini ] -> [ _SP ] [ MushroomLarge | MushroomMini StillViable ]
vertical [ _SP ] [ MushroomLarge | MushroomMini StillViable ] -> [ _SP ] [ MushroomLarge | MushroomLarge MushroomChanged random RandomGrowthParticles1 ]
[ _SP ] [ MushroomMini StillViable ] -> [ _SP ] [ MushroomMini ]
[ _SP ] [ MushroomLarge | Arable no Blocker ] -> [ _SP ] [ MushroomLarge | Arable MushroomMiniStraight MushroomChanged random RandomGrowthParticles1 ]

(Reeds growning and spreading)
[ _SP ] [ ReedYoung ] -> [ _SP ] [ ReedMatureStraight random RandomGrowthParticles1 ]
(Grow around the shoreline.)
[ _SP ] [ ReedMature Shore | Shore no Blocker ] -> [ _SP ] [ ReedMature Shore | Shore ReedYoung ]

[ _SP ] [ TomatoYoung ] -> [ _SP ] [ TomatoMatureStraight random RandomGrowthParticles1 ]

(Picking up)
[ action Player GrassResource no Carried ] -> [ Player GrassCarried ]
[ action Player ReedResource no Carried ] -> [ Player ReedCarried ]
[ action Player FlowerResource no Carried ] -> [ Player FlowerCarried ]
[ action Player KumaraResource no Carried ] -> [ Player KumaraCarried ]
[ action Player TomatoResource no Carried ] -> [ Player TomatoCarried ]
[ action Player MushroomResource no Carried ] -> [ Player MushroomCarried ]
[ action Player TreeResource no Carried ] -> [ Player TreeCarried ]

(If a goal is blocked, mark it as such.)
[ TreeGoal TreeGoalMarker Blocker no TreeBase ] -> [ TreeGoal TreeGoalMarkerBlocked Blocker ]

(Is a goal ready for a tree?)
[ GoalSurround Mature | GoalAdjacent Mature | GoalSurround Mature ] -> [ GoalSurround Mature | GoalAdjacent Mature GoalSurroundsReady | GoalSurround Mature ]
[ NoMushroomChanged ] [ TreeGoal no GoalReady no Blocker | no MushroomLarge ] -> [ NoMushroomChanged ] [ TreeGoal StillViable | ]
[ TreeGoal StillViable | no GoalSurroundsReady ] -> [ TreeGoal | ]
[ TreeGoal StillViable TreeGoalMarkerNotReady ] -> [ TreeGoal GoalReady TreeGoalMarkerReady1 ]

(Are all goals ready?)
[ Player ] -> [ Player AllGoalsReady ]
[ AllGoalsReady ] [ TreeGoal no GoalReady ] -> [ ] [ TreeGoal ]

[ AllGoalsReady ] [ RepairOnGoalsReady VerticalBridgeBroken ] -> [ AllGoalsReady ] [ VerticalBridgeRepairing1 ]
[ AllGoalsReady ] [ RepairOnGoalsReady HorizontalBridgeBroken ] -> [ AllGoalsReady ] [ HorizontalBridgeRepairing1 ]
[ AllGoalsReady ] -> [ ]

(If action and still carrying, probably can't do that thing here...)
[ action Player Carried no NeedToWaitSymbol ] -> [ action Player Carried CantPlantSymbol1 ]

(...but maybe you just need to wait. NeedToWaitSymbol replaces CantPlantSymbol on same layer.)
[ GrowingCrops ] [ action Player StandardArableCarried Arable no Blocker ] -> [ GrowingCrops ] [ action Player StandardArableCarried Arable NeedToWaitSymbol1 ]
[ GrowingCrops ] [ action Player ReedCarried Shore no Blocker ] -> [ GrowingCrops ] [ action Player ReedCarried Shore NeedToWaitSymbol1 ]
[ GrowingCrops ] [ action Player TreeCarried TreeGoal no Blocker ] -> [ GrowingCrops ] [ action Player TreeCarried TreeGoal NeedToWaitSymbol1 ]

[ GrowingCrops ] [ NeedToWaitSymbol ] -> [ GrowingCrops ] [ NeedToWaitSymbol NeedToWaitViable ]
[ NeedToWaitSymbol no NeedToWaitViable ] -> [ ]
[ NeedToWaitViable ] -> [ ]

[ > Player NeedToWaitSymbol ] -> [ > Player > NeedToWaitSymbol ]

(Walking)
[ > Player | Shore no Blocker ] -> [ > Player | Shore FootstepSoundShore ]
[ > Player | Earth no Blocker ] -> [ > Player | Earth FootstepSoundEarth ]
[ > Player | Arable Blocker ] -> [ > Player | Arable Blocker FootstepSoundPlant ]
[ > Player | Bridge ] -> [ > Player | Bridge FootstepSoundBridge ]
[ > Player | Stone ] -> [ > Player | Stone FootstepSoundStone ]

[ > Player Arable no Blocker no GrowthParticles ] -> [ > Player Arable random RandomGrowthParticles1 ]
[ < Player GrassMature ] -> [ < Player GrassMatureRustle1 ]
[ < Player ReedMature ] -> [ < Player ReedMatureRustle1 ]
[ < Player TomatoMature ] -> [ < Player TomatoMatureRustle1 ]
[ < Player KumaraMature ] -> [ < Player KumaraMatureRustle1 ]
[ < Player MushroomLarge ] -> [ < Player MushroomLargeRustle1 ]
[ < Player MushroomMini ] -> [ < Player MushroomMiniRustle1 ]
[ < Player Flower ] -> [ < Player FlowerRustle1 ]
[ > Player Carried ] -> [ > Player > Carried ]
[ > Player | Gravestone ] -> cancel
[ > Player | no Walkable ] -> cancel
