RULES

( -------- action rules -------- )

[ Action Player Water ] -> cancel
[ Action Player Wine ] -> cancel

[ Action Jesus Cross ] -> [ Jesus_wearing_Cross ]
[ Action Jesus Bread ] -> [ Jesus_wearing_Bread ]
[ Action Jesus_wearing_Cross no Bread ] -> [ Jesus Cross ]
[ Action Jesus_wearing_Bread Cross ] -> [ Jesus_wearing_Cross Bread ]
[ Action Jesus_wearing_Cross Bread ] -> [ Jesus_wearing_Bread Cross ]
(transformation de l'eau en vin)
[ Action Player | Water] -> [ Player | Wine ]
(multiplication des pains)
[ Action Jesus_wearing_Bread no Cross ] -> [ Jesus_wearing_Bread Bread ]


( -------- movement rules -------- )

[ > Jesus | Water ] -> cancel
[ > Jesus_wearing_Bread | Water ] -> cancel
[ > Jesus Drunkenness | Wine ] -> cancel
[ Global_marker no Global_walk_on_wine ][ > Jesus_wearing_Cross Drunkenness | Wine ] -> cancel

(Jesus ne devrait pas aller dans un trou avec sa croix.)
[ > Jesus_wearing_Cross | Hole ] -> cancel

[ Global_marker no Global_walk_on_wine ][ > Player | Wine  ] -> [ Global_marker no Global_walk_on_wine ][ > Player Drunkenness | ]

[ Global_marker Global_walk_on_wine ][ > Jesus | Wine  ] -> [ Global_marker Global_walk_on_wine ][ > Jesus Drunkenness | ]


( -------- late rules -------- )

late [ Player No Drunkenness ][ Drunkenness ] -> [ Player Drunkenness ][ ]

late [ No Player Drunkenness ] -> [ ]

late [ Wine | Water ] -> [ Wine | Wine ]

late random [ Ghost_Jesus | Cross ] -> [ | Ghost_Jesus Cross ] again
