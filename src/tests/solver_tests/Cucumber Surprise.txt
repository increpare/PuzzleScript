RULES

[moving player] [ push ] -> [ moving player ] [ ]

up [ push_up stationary katze | no obstacle ] -> [ | push_up action katze ] again
+ down [ push_down stationary katze | no obstacle ] -> [ | push_down action katze ] again
+ left [ push_left stationary katze | no obstacle ] -> [ | push_left action katze ] again
+ right [ push_right stationary katze | no obstacle ] -> [ | push_right action katze ] again


[ >  Player | gurke | no obstacle ] -> [  >  Player | > gurke | no obstacle  ] again


[ < player | katze ] -> [ < player | < katze ] again

late up [ gurke | katze | no obstacle] -> [ gurke | push_up katze | no obstacle] sfx0
