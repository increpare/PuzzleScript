======

[ prev1 | player ] -> [ > prev1 | player ]
[ prev2 | prev1 ] -> [ > prev2 | prev1 ]

late [ player 1 ] [ prev1 1 ] [ prev2 1] -> [ player 2 ] [ prev1 ] [ prev2 ] sfx1 again
late [ player 2 ] [ prev1 2 ] [ prev2 2] -> [ player 3 ] [ prev1 ] [ prev2 ] sfx2 again
late [ player 3 ] [ prev1 3 ] [ prev2 3] -> [ player 4 ] [ prev1 ] [ prev2 ] sfx3 again
