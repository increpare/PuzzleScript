RULES
======


[ > player | hole ] -> cancel

[ arrow ] [ player ] -> [arrow] [ action player ]
[ AU ball ] -> [  up ball ]
[ AD ball ] -> [  down ball ]
[ AL ball ] -> [  left ball ]
[ AR ball ] -> [  right ball ]

[ > ball | stationary ball ] -> [ stationary ball | > ball ] 

[ > ball | wall ] -> [ T ball | wall ]
[ > ball | T ] -> [ T ball |  ]

[ > Player | ball ] -> [ > Player | > ball ] sfx0 
[ > ball | ball ] -> [ > ball | > ball ]
