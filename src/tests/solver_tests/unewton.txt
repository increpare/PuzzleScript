=======
LEGEND
=======

. = Background
# = Wall
P = Player
C = ball
@ = ball and target
O = target
Arrow = AU or AD or AL or AR
Obstacle = ball or Wall
block = T or wall
h = hole
$ = ball and hole
f = target and hole
k = ball and hole and target

======
RULES
======


[ > player | hole ] -> cancel

[ arrow ] [ player ] -> [arrow] [ action player ]
[ AU ball ] -> [ AU up ball ]
[ AD ball ] -> [ AD down ball ]
[ AL ball ] -> [ AL left ball ]
[ AR ball ] -> [ AR right ball ]

[ > ball | ball ] -> [ > ball | > ball ] 

[moving ball arrow] [ moving ball no arrow ] -> [ stationary ball ] [ moving ball no arrow ]

[arrow]->[]

[ > ball | wall ] -> [ T ball | wall ]
[ > ball | T ] -> [ T ball |  ]
[t]->[]

[ stationary ball | > ball ] -> sfx2

[ action Player | ball ] -> [ action Player | > ball ] sfx0 

[ > Player | ball ] -> [ > Player | > ball ] sfx0 
[ > ball | ball ] -> [ > ball | > ball ]

up 		[ action player ] [ > ball | no obstacle ] -> [ action player ] [ | ball AU ] again
down    [ action player ] [ > ball | no obstacle ] -> [ action player ] [ | ball AD ] again
left 	[ action player ] [ > ball | no obstacle ] -> [ action player ] [ | ball AL ] again
right 	[ action player ] [ > ball | no obstacle ] -> [ action player ] [ | ball AR ] again

WINCONDITIONS
