down [ action elephantA_4 | log | no log ] -> [ action elephantA_4 | airlog | ] sfx1
(cancel retraction if overlaps)
right [ action elephantA_4 ] [ log | elephantA_11 ] -> cancel sfx0

(retract trunk)
right [ action elephantA_4 ] [  | elephantA_11 ] -> [ elephantA_4 ]  [ elephantB_11A | elephantB_11B ]

(extend trunk)
right [ action elephantA_4 ] [  elephantB_11A | elephantB_11B ] -> [ action elephantA_4 ]  [  | elephantA_11 ]
(drop log if extending trunk)
down [ action elephantA_4 | airlog | no obstacle  ] -> [  elephantA_4 | | log  ] sfx2
down [ action elephantA_4 | airlog | obstacle  ] -> cancel sfx0
