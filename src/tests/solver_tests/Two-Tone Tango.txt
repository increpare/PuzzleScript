======
RULES
======

[ up player | Freund ] -> [ up player | up Freund ]
[ down player | Freund ] -> [ down player | down Freund ]
[ left player | Freund ] -> [ left player | left Freund ]
[ right player | Freund ] -> [ right player | right Freund ]


vertical [ up freund | stationary freund ] -> [ up freund | up freund ]
vertical [ down freund | stationary freund ] -> [ down freund | down freund ]
vertical [ left freund | stationary freund ] -> [ left freund | left freund ]
vertical [ right freund | stationary freund ] -> [ right freund | right freund ]

(forward propagation)
[ >  pusher | Crate ] -> [  >  pusher | > Crate  ]


(backwards propagation)

[ > pusher | stationary blocker ] -> [ stationary pusher | stationary blocker ]
+ vertical [ up freund | stationary freund ] -> [ stationary freund | stationary freund ]
+ vertical [ down freund | stationary freund ] -> [ stationary freund | stationary freund ]
+ vertical [ left freund | stationary freund ] -> [ stationary freund | stationary freund ]
+ vertical [ right freund | stationary freund ] -> [ stationary freund | stationary freund ]

(again forwards propagation)
[ stationary pusher | > crate ] -> [ stationary pusher | stationary crate ]

[stationary player ]->  cancel

[ < player | stationary freund ] -> sfx1
[ > player | | stationary freund ] -> sfx2

WINCONDITIONS
