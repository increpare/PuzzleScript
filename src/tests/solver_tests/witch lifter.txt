======
RULES
======


[ movable ] -> [ movable movable_src ]

[  player ] -> again

up [ up player_s2 ladder | no ladder ] -> cancel

[ stationary player ] [ witch ] -> [ stationary player ] [ down witch ]

(if player shrinking, apply gravity)

[ down player_s2 ] [ witch ] -> [ down player_s2 ] [ down witch ]

 [ horizontal player ] -> [ horizontal player player_source direct_push ]

[ up Player_short no ladder ] -> [ up Player_short move_up ]
up [ up Player_short ladder | no ladder ] -> [ up Player_short move_up ladder | ]

up [ obstacle | down Player_S2 | down Player_S1  ] -> [ obstacle| Player_short |   ]


[ stationary player no ladder ] -> [ down player ]

(propagate direct_push)
[ > pusher direct_push | pusher ] -> [ > pusher direct_push | > pusher direct_push ]
+ [ orthogonal w1 direct_push| stationary w1 ] -> [ orthogonal w1 direct_push| orthogonal w1 direct_push]
+ [ orthogonal w2 direct_push| stationary w2 ] -> [ orthogonal w2 direct_push| orthogonal w2 direct_push]
+ [ orthogonal w3 direct_push| stationary w3 ] -> [ orthogonal w3 direct_push| orthogonal w3 direct_push]
+ [ orthogonal w4 direct_push| stationary w4 ] -> [ orthogonal w4 direct_push| orthogonal W4 direct_push]


[ stationary player ladder ] -> [stationary player ladder player_on_ladder ]
(propagate movements - first through different objects)
[ > pusher | pusher no player_on_ladder] -> [ > pusher | > pusher ]
(then within objects)
+ [ orthogonal w1 | stationary w1 ] -> [ orthogonal w1 | orthogonal w1 ]
+ [ orthogonal w2 | stationary w2 ] -> [ orthogonal w2 | orthogonal w2 ]
+ [ orthogonal w3 | stationary w3 ] -> [ orthogonal w3 | orthogonal w3 ]
+ [ orthogonal w4 | stationary w4 ] -> [ orthogonal w4 | orthogonal w4 ]
(you can propagate motions upwards between objects)
+ up [ orthogonal pusher | stationary pusher ] -> [ orthogonal pusher | orthogonal pusher ]


[player_on_ladder]->[]

(vertical pushing is hard constraint)
up [ > movable | solidtile ] -> cancel

(now back-propagate non-movements - only need to do this horizontally, or for gravity, or upwards if its a soft push)
horizontal [ > movable | stationary obstacle] -> [ movable | stationary obstacle ]
+ down [ > movable | stationary collidable] -> [ movable | stationary collidable ]
+ down [ horizontal movable no direct_push | stationary collidable] -> [ movable | stationary collidable ]
	(back-propagate within objects)
+ [ orthogonal w1 | stationary w1 ] -> [ stationary w1 | stationary w1 ]
+ [ orthogonal w2 | stationary w2 ] -> [ stationary w2 | stationary w2 ]
+ [ orthogonal w3 | stationary w3 ] -> [ stationary w3 | stationary w3 ]
+ [ orthogonal w4 | stationary w4 ] -> [ stationary w4 | stationary w4 ]
+ [ orthogonal player | stationary player ] -> [ stationary player | stationary player ]
(should stationaryness propagate upwards? i think so? - IF the motion is horizontal and not a direct push)
+ up [ stationary pusher | horizontal witch no direct_push ] -> [ stationary pusher | stationary witch ]
+ down [ down pusher | stationary collidable ] -> [ stationary pusher | stationary collidable ]


[ > w1 | stationary obstacle ] -> [ w1 | stationary obstacle ]

up [ > pusher | solidtile ] -> cancel

down [ moving player | moving witch ] -> cancel

[ moving witch decoration ] -> [ moving witch moving decoration ]

late up [  move_up Player_short ] -> cancel
late up [  move_up | Player_short ] -> [ Player_S2 | Player_S1]

late [ player_source player ] -> cancel

late [ player_source ] -> []

late [ direct_push ] -> [ ]
late [ player_head ] -> [ player_head has_won ]
late down [has_won] [ witch | solidtile ]  -> [ ] [ witch|solidtile]
late [ has_won ] [ movable_src no movable ] -> [ ] [ movable_src ]
late [ movable_src ] -> []
late [has_won ] [ eye] -> [ has_won] [ deadeye]
late [has_won] -> [] win
