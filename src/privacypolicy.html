PuzzleScript Privacy Policy

If you're playing PuzzleScript games hosted on puzzlescript.net

Puzzlescript games accesed at urls like "puzzlescript.net/play.html?..."" are stored on github as gists and fetched from there with their Developer API. Their privacy policy is here.

Why do I need to store things there? I need to store games somewhere - I think the sharing functionality is really importat for puzzlescript, and makes making + sharing games really easy.

If you don't want to use the online-sharing facilities you can still save/load game locally, and can still export standalone html files.

The puzzlescript player uses google analytics - I use this to track what games people have been playing, so as to be able to more easily keep track of what people are making with the tool.

If you're creating PuzzleScript games

  • Puzzlescript can save things to your browser's localstorage - it's just used for loading/saving files in the file save dropdown.
  • If you share a game, you need to authorize with your github account. Then, when you hit share in future, a publicly-accessible copy of your game is uploaded to a gist in your account.
If you have any questions about this, hit me up at analytic@gmail.com.
